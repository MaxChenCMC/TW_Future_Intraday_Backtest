以2014到2019`台指期連續月`的日k棒為分析對象

設計11個k棒漲跌排列的特徵

先用隨機森林算出哪個特徵值較具顯著性

再逐一畫出每個特徵值為交易訊號時的各自累計損益

並將停損考慮進去以優化績效
